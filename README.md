# 모두어리
모두어리는 교환일기 컨셉의 웹서비스입니다  
현재 개발중이며 아래는 최종 결과물의 예상 이미지 입니다  

![sketch00](https://user-images.githubusercontent.com/80666066/116812151-33545980-ab88-11eb-829a-4f1c70a5a67a.png)


### 기술 스택
- Java, Spring Boot
- JPA
- H2 database  
- Thymeleaf

</br>
